<h2>How to run:</h2>

<h3>Clone the repository:</h3>

```
$ git clone https://github.com/vivirortega/cotabox.git
```

<h3>Go to front-end folder:</h3>

```
$ cd front-end
```


<h3>Install dependencies:</h3>

```
$ npm or yarn install
```

<h3>Initializing the API:</h3>

```
$ npm start
```
